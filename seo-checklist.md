# SEO Checklist para AliTransfers

## Elementos Técnicos
- [x] Título optimizado con palabras clave
- [x] Meta descripción relevante y atractiva
- [x] Meta keywords (opcional, pero incluidos)
- [x] Etiqueta canonical en cada versión de idioma
- [x] Meta robots (index, follow)
- [x] Sitemap.xml creado y actualizado
- [x] robots.txt configurado correctamente
- [x] Etiquetas hreflang y alternate para multilenguaje
- [x] Open Graph y Twitter Card para redes sociales
- [x] Datos estructurados LocalBusiness (JSON-LD)

## Contenido y Accesibilidad
- [x] Solo un h1 por página, jerarquía de encabezados correcta
- [x] Atributos alt descriptivos en imágenes
- [x] Lazy loading en imágenes principales
- [x] Roles ARIA y atributos de accesibilidad en elementos clave
- [x] Enlaces internos entre secciones relevantes

## Experiencia de Usuario
- [x] Favicons y Apple Touch Icon presentes
- [x] Diseño responsive (Bootstrap)
- [x] Botón de WhatsApp y enlaces de contacto visibles
- [x] Contenido relevante y orientado al usuario

## Recomendaciones Futuras
- [ ] Revisar y mantener sitemap.xml y robots.txt tras cambios
- [x] Añadir datos estructurados para reseñas/testimonios
- [ ] Revisar velocidad de carga y optimización de imágenes adicionales
- [ ] Auditar enlaces rotos periódicamente
- [ ] Revisar accesibilidad avanzada (contraste, navegación por teclado, etc.)
- [ ] Mantener actualizadas las etiquetas hreflang si se agregan más idiomas
